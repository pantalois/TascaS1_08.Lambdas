Task S1.08.Lambdas

📄 Description

This project consists of developing and practicing the concept of Lambdas in Java. Lambdas are very useful, they allow us to define and assign an implementation
of a single method (from a functional interface) to a variable, which can make our code cleaner and more flexible.

💻 Technologies Used

Java (recommended version: 17 or higher) Recommended IDE: IntelliJ IDEA, Eclipse, or VS Code with Java extension Git

📋 Requirements

Java JDK 17+ Git A Java-compatible IDE

🛠️ Installation

Clone this repository: git clone https://github.com/pantalois/TascaS1_08.Lambdas.git

Navigate to the project directory: cd https://github.com/pantalois/TascaS1_08.Lambdas.git

Open the project with your preferred IDE.

▶️ Execution

Compile and run the Main.java files for the exercise you are working on to see the execution.
